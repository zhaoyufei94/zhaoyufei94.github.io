LibTorch使用记录

发表于 2022-01-16

为了快速在C++算法引擎中验证PyTorch模型的效果，尝试集成libtorch，仅在Linux服务器上加载模型并进行前向运算

下载安装

从PyTorch官网下载对应版本并解压，如最新版本不匹配CUDA版本，可以从这里找到历史版本，为了防止该博客无法访问，将内容粘贴至 Appendices 中

如果要使用CUDA，则必须与机器的CUDA版本完全一致，小版本号不一样也可能无法启用CUDA

cmake中添加

就可以引用torch的头文件了

吐槽一下，libtorch的头文件很多很乱，无法直接通过 include_directories 找到对应的头文件，IDE中就很难跳转查看相应接口

用到的API接口

主要用到的：

  1. 加载.pt模型
  2. vector<float> 与 at::Tensor 之间相互转换
  3. 模型前向计算

注：

  1. 模型在 Python 中通过 ScriptModule.save() 接口进行保存

  2. module.eval() 经测试是不产生实际效果的，需要在 Python 中保存模型前就先设置为 eval 模式

  3. 还没研究怎么通过 module 的接口获取到输入tensor的维度信息
